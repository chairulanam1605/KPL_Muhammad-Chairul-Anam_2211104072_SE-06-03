### 3. Penjelasan Code
##### Program ini merupakan implementasi antarmuka grafis sederhana menggunakan JavaScript dan HTML yang bertujuan untuk mengidentifikasi tanda dari suatu bilangan bulat yang dimasukkan oleh pengguna. Dalam program ini terdapat sebuah form dengan satu input teks (textbox) untuk memasukkan angka, sebuah tombol (button) untuk menjalankan proses, serta satu label untuk menampilkan hasil. Inti dari program ini terletak pada method bernama CariTandaBilangan() yang menerima satu parameter bertipe integer dan mengembalikan nilai berupa string. Metode ini memiliki tiga aturan logika: apabila nilai kurang dari nol maka akan mengembalikan “Negatif”, jika lebih dari nol maka mengembalikan “Positif”, dan jika sama dengan nol maka hasilnya adalah “Nol”. Ketika pengguna memasukkan angka ke dalam textbox dan menekan tombol, method CariTandaBilangan() akan dipanggil dengan nilai input tersebut, dan hasil akhirnya akan ditampilkan melalui label output. Program ini juga dilengkapi dengan kode unit test untuk memastikan bahwa seluruh percabangan dalam method telah diuji secara menyeluruh. Implementasi ini memperlihatkan bagaimana JavaScript dapat digunakan untuk membangun interaksi pengguna secara langsung melalui browser sekaligus menerapkan pengujian logika dasar guna menjaga keandalan fungsi yang dibuat.
